# Myotis Bat Biodiversity Analysis

This project investigates how climatic factors, specifically mean temperature and temperature seasonality, influence species richness in bats of the *Myotis* genus across global geographic regions.

Species occurrence data from the Barcode of Life Data System (BOLD) is integrated with climate data from WorldClim to explore biodiversity patterns. The analysis includes data cleaning, spatial visualization, and statistical modeling (linear regression, ANOVA, and non-parametric tests).

Initial results suggest higher species richness in warmer regions and a significant relationship between temperature variability and biodiversity, supporting ecological theories such as the latitudinal diversity gradient.

This repository is being further developed to incorporate genetic diversity and phylogenetic analysis using next-generation sequencing data.
